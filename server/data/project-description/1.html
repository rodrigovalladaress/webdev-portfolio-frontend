The Content Insight Tool is a system that I built for Hub. It retrieves data from videos of different social media and shows them in a dashboard. There you can compare videos from different brands, social media platforms and periods of time.

The tool consists of different modules:

  • Dashboard: A Vue application that is based around the Quasar framework. The charts are built using ECharts.
  • Backend: A Node.js Express server. It uses Sequelize to connect to a Postgres database.
  • Data Fetching Tool: A Node.js server that connects to several social media APIs to update the videos of the system. The API calls are cached on a MongoDB database.
