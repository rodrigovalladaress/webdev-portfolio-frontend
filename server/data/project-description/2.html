Ahamo was a webapp that let people save notes of an event.

Basically, attendants added notes to a timestamp during a live event (or an on-demand video after the event has finished). The notes were public, and the application showed a graph with the number of notes on each timestamp. It was a great way of knowing which part of the event was most most interesting for the attendants.

The notes could hold different data:

  • Text
  • Images
  • Audio
  • Video

The backend of the application was a Go server. The frontend was based on React, and used Chart.js to build the note charts.
